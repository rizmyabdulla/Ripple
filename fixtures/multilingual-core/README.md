# Multilingual-core fixture

Synthetic, consent-tagged speaker-disjoint manifests used to exercise Phase 1 tooling
without claiming rights to any real speech corpus.

Languages covered: `en-US`, `es-ES`, `fr-FR`, `de-DE`, `hi-IN`, `ja-JP`.

These JSON manifests pin checksums and provenance. Replace with licensed corpora through the
dataset card and exclusion-log process before any public training or release.
